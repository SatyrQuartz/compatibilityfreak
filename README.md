=============================

Installers and Scripts for some Compatibility Layers

=============================

### emuroot.sh

    linux-arm64 (Debian)
	linux-armel (Debian)
	linux-armhf (Debian)
	linux-i386 (Debian)
	linux-mips (Debian)
	linux-mips64el (Debian)
	linux-mipsel (Debian)
	linux-powerpc (Void Linux PPC)
	linux-powerpc64 (Void Linux PPC)
	linux-powerpc64le (Debian)
	linux-s390x (Debian)
	linux-x86_64 (Debian)

### install_houdini32.sh

Installs Houdini (32-bit ARM Emulator) on amd64 or i386 PC


### install_houdini64.sh

Installs Houdini (64-bit ARM Emulator) on amd64 or i386 PC


### uninstall_houdini.sh

Removes Houdini on amd64 or i386 PC
